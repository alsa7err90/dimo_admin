
## install
<p>1- git clone https://github.com/alsa7err90/dimo_admin.git</p>
2- composer install
3- cp .env.example .env
4-php artisan key:generate
5-Set your database credentials in your .env file
6-php artisan migrate:fresh --seed
7-npm install
8-npm run dev
9-php artisan serve
